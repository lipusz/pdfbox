Decrypt

  • Description

Description

This application will decrypt a PDF document.

Note
You must have the owner password to decypt the document!
Command Line Parameter Type Default Value Description
<password> string None The owner password to the PDF.
<inputfile> filename None The PDF file to decrypt.
[outputfile] filename The input file The file to save the decrypted document to. If left blank then it will be the same as the input file.
